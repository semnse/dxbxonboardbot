# 📋 ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ БОТА

## Процесс работы

### 1. Создание чата с клиентом

1. Внедренец создаёт групповой чат в Telegram с клиентом
2. Добавляет бота **@Carrot1_test_bot** в чат

### 2. Привязка карточки Bitrix

Внедренец пишет в чат команду:

```
/add <ID карточки>
```

**Где взять ID:**
- Откройте карточку в Bitrix24
- Скопируйте ID из ссылки:
  ```
  https://docsinbox.bitrix24.ru/company/personal/.../crm/leader/1070/9200/
                                                                  ^^^^
                                                                  ID
  ```

**Пример:**
```
/add 9200
```

**Ответ бота:**
```
✅ Карточка привязана!

Компания: ООО "Гастромаркет" 9704247610
ID Bitrix: 9200
Стадия: Ждём действий клиента - Пауза до вывода

Я буду отправлять отчёты в этот чат каждое утро в 9:00 МСК.
```

### 3. Автоматические отчёты

**Каждое утро в 9:00 МСК** бот отправляет в чат отчёт:

```
🔍 ООО "Гастромаркет", напоминаем о шагах!

✅ УЖЕ ДОСТУПНО:
• ЕГАИС
• Накладные
• ЮЗЭДО
• Меркурий
• Маркировка

⏳ ОСТАЛОСЬ СДЕЛАТЬ:
• Заполнить и выгрузить номенклатуру в учетной системе
• Подтвердить легальность площадки в Меркурии

💡 ЭТО ВАЖНО, ПОТОМУ ЧТО:
Без этих шагов вы не сможете передавать чеки в ЕГАИС.
Это может привести к штрафам от Росалкогольрегулирования.
```

### 4. Ручной запрос отчёта

В любой момент можно запросить текущий отчёт:

```
/report
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкция |
| `/add <ID>` | Привязать карточку Bitrix к чату |
| `/report` | Получить текущий отчёт |
| `/help` | Справка по командам |
