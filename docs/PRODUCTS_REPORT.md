## 🛠️ Реализация

### Файл: `app/services/bitrix_product_service.py`

#### Структура данных

```python
class BitrixProductService:
    # Смарт-процесс "Внедрение"
    PRODUCT_ENTITY_ID = 1056
    
    # Маппинг продуктов → ID карточек
    PRODUCT_CARD_IDS = {
        'usedo': 68566,       # ЮЗЭДО
        'marking': 68564,     # Маркировка
        'nakladnye': 68562,   # Накладные
        'mercury': 68560,     # Меркурий
        'egais': 68558,       # ЕГАИС
    }
    
    # Маппинг стадий → сообщения (50+ стадий)
    STAGE_MESSAGES = {
        'usedo_2248': {...},  # ЮЗЭДО: Новая заявка
        'egais_2916': {...},  # ЕГАИС: Сопоставление
        # ... все стадии
    }
```

#### Методы

1. **`get_product_status(product_card_id)`**
   - Получает статус продукта по ID карточки
   - Возвращает: `{'product_code', 'card_id', 'stage_id', 'item'}`

2. **`get_all_products_status(parent_deal_id)`**
   - Получает статусы всех продуктов для торговой точки
   - Возвращает: `{'usedo': {...}, 'marking': {...}, ...}`

3. **`format_report(products_status)`**
   - Формирует текстовый отчёт
   - Возвращает: Markdown текст с инструкциями

## 🔄 Интеграция с ботом

### Добавление команды `/product_report`

```python
# app/bot/subscriptions.py
from app.services.bitrix_product_service import bitrix_product_service

@subscriptions_router.message(Command("product_report"))
async def cmd_product_report(message: Message):
    """Отчёт по продуктам"""
    
    # Получаем ID торговой точки из БД
    bitrix_deal_id = "18306"  # Из БД
    
    # Получаем статусы продуктов
    products_status = await bitrix_product_service.get_all_products_status(bitrix_deal_id)
    
    # Формируем отчёт
    report = bitrix_product_service.format_report(products_status)
    
    await message.answer(report, parse_mode="Markdown")
```

## 🔧 Расширение для других торговых точек

Для добавления другой торговой точки:

1. Узнайте ID карточек продуктов в смарт-процессе "Внедрение"
2. Добавьте в `PRODUCT_CARD_IDS`:

```python
# Для ООО "Ромашка" (карточка 12345)
PRODUCT_CARD_IDS_ROMASHKA = {
    'usedo': 11111,
    'marking': 22222,
    'nakladnye': 33333,
    'mercury': 44444,
    'egais': 55555,
}
```

3. Сохраните маппинг в БД для автоматического определения
